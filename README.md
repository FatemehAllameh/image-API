# Image API Project

A simple web application for searching and displaying images using the Unsplash API.

> Note: Users accessing from Iran may need VPN to fetch data properly.

You can try the project online [here](https://image-api-bay.vercel.app/)

## Features

- Search images by keyword
- Display results with image previews
- Pagination for navigating multiple pages of results
- Loading indicators while fetching data
- Warning messages for invalid or empty searches
- Responsive layout for all screen sizes

## Technologies Used

- HTML
- CSS
- JavaScript
- Unsplash API

## How to Use

1. Download the project  
   Click the green "Code" button on GitHub and choose "Download ZIP", then extract the folder.

2. Open the project  
   Navigate to the extracted folder and open the index.html file in your browser.

3. Use the Image App  
   - Search for images using the input field.  
   - Navigate through search results using pagination buttons.

Or, you can try the project online [here](https://image-api-bay.vercel.app/)
